TBF Secure File Management System — User Guide
1. What this application is for
The TBF Secure File Management System is a company file manager for Two Brothers Freight. It lets employees:

Browse and work with shared company folders based on their team (group) permissions
Store private files in My Files (personal space, visible only to them)
Search across files they can access
Star favorites and see recent activity on the dashboard
Administrators manage users, groups, permissions, and audit logs. The app is accessed in a web browser; accounts are created by administrators—not through public self-registration.

2. Complete user workflow
First-time / daily flow
Sign in with your company email and password on the login page.
You land on the Dashboard — your home base with shortcuts, storage usage, favorites, and recent files.
Shared work: go to Folders → open a folder → browse subfolders and files → upload, preview, download, or delete (depending on your permissions).
Private work: go to My Files → create subfolders if needed → upload and manage your personal files.
Use the header search anytime to find files by name across accessible folders.
Star (★) folders or files you use often; find them on the Dashboard under Favorites.
Logout from the header when finished.
Session behavior you should know
Only one active session per account: signing in on another device signs you out elsewhere.
After 30 minutes of inactivity, you may be signed out automatically.
If you see a message that your account was used on another device, sign in again.

6. Concepts explained for users
Folders (shared)
Root folders are top-level shared areas (e.g. folder1, folder2).
Subfolders nest inside them for organization.
You only see root folders your group has access to.
Empty subfolders must be deleted before parent folders can be removed (where delete is allowed).
My Files
Completely private — other users cannot see your My Files.
50 GB total storage, 50 MB max per file.
You can organize with subfolders like shared folders.
Files
Every uploaded file must have a file extension (e.g. .pdf, .docx).
.zip and .bat files are blocked and cannot be uploaded.
Preview works for common images, PDFs, and videos; other types can only be downloaded.
Users
Each person has a name, email, role, and optional group.
Without a group, regular users cannot access shared folders (Dashboard will suggest contacting admin).
Groups
Teams like eld, dispatch, safety, maintenance (examples from default setup).
Each user belongs to one group (or none).
The group determines which folders you see and what you can do there.
Permissions (group ACL)
Permissions are assigned per group, per folder (and optionally a specific subfolder). They control shared folder behavior—not My Files, and not admin screens.

7. Roles and what each can do
USER (regular employee)
Dashboard, Folders (per group ACL), My Files, global search, favorites
Upload/download/delete/preview only as allowed on each shared folder
Cannot manage users, groups, logs, or root folders
Must be assigned to a group to access shared folders
ADMIN
Everything a USER can do, plus:

Users — create and manage regular USER accounts; assign groups; reset user passwords; delete users
Audit Logs — view activity of regular users only
Full access to all shared folder actions in the UI (upload/download/delete/subfolder management) regardless of group checkboxes
Cannot access Groups page, System Admin page, create super admins, or manage admin accounts (limited by super admin)
SUPER_ADMIN
Everything an ADMIN can do, plus:

Groups — create/delete groups; configure all permission checkboxes
Users — create ADMIN accounts; manage all users except deleting super admin
Folders list — create, rename, delete root folders
Audit Logs — see all roles (filter by User/Admin/Super Admin)
System Admin overview page
Unrestricted access to shared folders
Important: For shared files, your group permissions matter for regular USER accounts. Roles mainly control management screens and admin bypass on shared folders—not My Files.

10. Validations, restrictions, and important behaviors
Email & accounts
Company emails must end with @twobrothersfreight.com (except configured super admin login).
Invalid or non-company emails are rejected when creating users.
Files & uploads
Max 50 MB per file (shared and My Files).
My Files total quota: 50 GB (shown on Dashboard).
Files must have an extension.
.zip and .bat uploads are blocked.
Upload shows progress; failed uploads can be retried.
Folders
Cannot delete folders that still contain files or subfolders.
Root folder delete/rename: super admin only.
Security & sessions
One login at a time per account.
~30 minutes inactivity may sign you out.
All shared file actions are permission-checked on the server—even if the button were visible, denied actions would fail.
Audit logs
No logs until you select at least one filter.
Changing any filter automatically refreshes results.
Search
Header search: all accessible shared files by name.
In-folder search: only files already loaded in the current folder view (client-side filter).

12. Inconsistencies & confusing UX (improvement opportunities)
EDIT and MOVE permissions are shown when configuring groups, but users cannot move files, and EDIT does not match “edit files” in everyday use.
Register page exists but is unreachable; all provisioning is under Users — the login page correctly says admins provision accounts.
System Admin appears on the Dashboard but not in the top navigation.
Admin bypass: Admins see full folder management UI even when group ACL says otherwise — fine for admins, but different from regular users.
Subfolder management on shared folders is role-based (admin), not permission-based (EDIT/DELETE checkboxes) — group permissions don’t control “create subfolder” for normal users.
Logs “Move” and “Edit” categories may rarely appear because those actions aren’t performed by the app today.
Permission badges on Folders page include EDIT/MOVE even when those actions aren’t available to users in the file UI.
My Files subfolders can be managed by all users, while shared folder subfolders only by admins — similar sidebar, different rules (may confuse).
Single-session logout may not be instant on the old device until the app detects the session change (depends on activity/API calls).
Dashboard “Storage Usage” applies only to My Files, not shared company folders.
